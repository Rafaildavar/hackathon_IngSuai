### **Промпт для создания сайта "Hackathon_Ing_SUAI"**

**Название проекта:** Hackathon_Ing_SUAI Document Processor

**Краткое описание:**
Создать современное веб-приложение для обработки документов через встроенную ИИ-модель в рамках хакатона Ингушского ГУ. Пользователь загружает изображения, PDF-файлы или папку с файлами, система обрабатывает их локально с помощью AI-модели на сервере и возвращает обработанные файлы для скачивания.

---

### **1. Функциональные требования (Что должен делать сайт):**

**Бэкенд (Server-Side Logic):**
*   **Эндпоинт для загрузки файлов:** `POST /api/upload`
    *   Принимает `multipart/form-data`.
    *   Поддерживает загрузку: одного файла (`image/*`, `.pdf`), нескольких файлов или `.zip`-архива.
    *   Сохраняет файлы во временную папку `uploads/`.
*   **Встроенная AI-обработка:**
    *   После загрузки файлов запускается процесс AI-обработки **на этом же сервере**.
    *   Для работы с PDF: конвертация PDF в изображения (если нужно).
    *   Вызов AI-модели для каждого изображения/документа.
    *   Обработка через систему очередей для стабильной работы.
*   **Эндпоинт для проверки статуса:** `GET /api/status/:task_id`
    *   Создает фоновую задачу для каждой обработки и возвращает `task_id`.
    *   Фронтенд периодически опрашивает этот эндпоинт для отслеживания прогресса.
*   **Эндпоинт для скачивания:** `GET /api/download/:task_id`
    *   После успешной обработки позволяет скачать результат.
    *   Если исходных файлов было несколько - запаковывает в `.zip`.

**Фронтенд (User Interface):**
*   **Главный экран:**
    *   Современный минималистичный дизайн с брендингом хакатона.
    *   Drag & Drop зона для загрузки файлов.
    *   Кнопка "Выберите файл(ы)".
    *   Подсказка: "Перетащите сюда фото документа, PDF или папку с файлами для обработки ИИ."
*   **Процесс работы:**
    *   Превью загруженных файлов.
    *   Индикатор прогресса загрузки.
    *   Анимированный индикатор обработки "ИИ обрабатывает документы...".
    *   Кнопка "Скачать результат" после завершения.

**Требования к обработке:**
